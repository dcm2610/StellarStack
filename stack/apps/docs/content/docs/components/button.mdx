# Button

A versatile button component built with Radix UI Slot and class-variance-authority.

## Import

```tsx
import { Button } from '@workspace/ui/components/button';
```

## Usage

```tsx
<Button>Click me</Button>
```

## Variants

```tsx
<Button variant="default">Default</Button>
```

## With Icons

```tsx
import { Play, Square, RotateCw } from 'lucide-react';

<Button>
  <Play /> Start
</Button>

<Button variant="destructive">
  <Square /> Stop
</Button>

<Button variant="outline">
  <RotateCw /> Restart
</Button>
```

## As Child

Use `asChild` to render as a different element:

```tsx
<Button asChild>
  <a href="/docs">Documentation</a>
</Button>
```

## Disabled State

```tsx
<Button disabled>Disabled</Button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'destructive' \| 'outline' \| 'secondary' \| 'ghost' \| 'link'` | `'default'` | Visual variant |
| `size` | `'default' \| 'sm' \| 'lg' \| 'icon' \| 'icon-sm' \| 'icon-lg'` | `'default'` | Button size |
| `asChild` | `boolean` | `false` | Render as child element |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | - | Additional classes |
