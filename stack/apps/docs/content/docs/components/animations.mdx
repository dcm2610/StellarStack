# Animation Components

StellarStack includes various animation components built with Framer Motion.

## Available Animations

| Component | Description |
|-----------|-------------|
| `FadeIn` | Fade in on mount |
| `GlowCard` | Card with glowing effect |
| `GradientText` | Animated gradient text |
| `FloatingParticles` | Background particle effect |
| `AnimatedNumber` | Number counting animation |
| `ProgressRing` | Circular progress indicator |
| `PulsingDot` | Status indicator dot |
| `NoiseOverlay` | Subtle noise texture |
| `DropZone` | Drag and drop area |

## Import

```tsx
import { FadeIn } from '@workspace/ui/components/shared/Animations/FadeIn';
import { GlowCard } from '@workspace/ui/components/shared/Animations/GlowCard';
import { AnimatedNumber } from '@workspace/ui/components/shared/Animations/AnimatedNumber';
```

Or import all:

```tsx
import * as Animations from '@workspace/ui/components/shared/Animations';
```

## FadeIn

Fades content in when mounted.

```tsx
<FadeIn>
  <Card>Content fades in</Card>
</FadeIn>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `delay` | `number` | `0` | Delay in seconds |
| `duration` | `number` | `0.5` | Duration in seconds |
| `direction` | `'up' \| 'down' \| 'left' \| 'right'` | `'up'` | Animation direction |

```tsx
<FadeIn delay={0.2} direction="left">
  <p>Slides in from left</p>
</FadeIn>
```

## GlowCard

Card with animated glow effect on hover.

```tsx
<GlowCard>
  <h3>Glowing Card</h3>
  <p>Hover to see the glow effect</p>
</GlowCard>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'primary'` | Glow color |
| `intensity` | `number` | `0.5` | Glow intensity |

## GradientText

Text with animated gradient.

```tsx
<GradientText>
  Animated Gradient
</GradientText>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `colors` | `string[]` | Default gradient | Gradient colors |
| `speed` | `number` | `3` | Animation speed (seconds) |

```tsx
<GradientText colors={['#ff0080', '#7928ca', '#ff0080']}>
  Custom Gradient
</GradientText>
```

## AnimatedNumber

Animates number changes.

```tsx
<AnimatedNumber value={1234} />
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | Number to display |
| `duration` | `number` | `0.5` | Animation duration |
| `format` | `(n: number) => string` | - | Number formatter |

```tsx
<AnimatedNumber
  value={1234.56}
  format={(n) => n.toFixed(2)}
/>
```

## ProgressRing

Circular progress indicator.

```tsx
<ProgressRing progress={75} />
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `progress` | `number` | - | Progress (0-100) |
| `size` | `number` | `48` | Ring size in pixels |
| `strokeWidth` | `number` | `4` | Stroke width |
| `color` | `string` | `'primary'` | Ring color |

```tsx
<ProgressRing
  progress={75}
  size={64}
  strokeWidth={6}
  color="green"
/>
```

### Props

| Prop | Type | Description |
|------|------|-------------|
| `status` | `'online' \| 'offline' \| 'warning' \| 'error'` | Status type |

## FloatingParticles

Background particle effect.

```tsx
<div className="relative">
  <FloatingParticles count={50} />
  <Content />
</div>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `count` | `number` | `30` | Number of particles |
| `color` | `string` | `'white'` | Particle color |
| `speed` | `number` | `1` | Animation speed |

## NoiseOverlay

Subtle noise texture overlay.

```tsx
<div className="relative">
  <NoiseOverlay opacity={0.05} />
  <Content />
</div>
```

## DropZone

Drag and drop upload area.

```tsx
<DropZone
  onDrop={(files) => handleUpload(files)}
  accept={['image/*', '.pdf']}
>
  Drop files here or click to upload
</DropZone>
```

### Props

| Prop | Type | Description |
|------|------|-------------|
| `onDrop` | `(files: File[]) => void` | Drop handler |
| `accept` | `string[]` | Accepted file types |
| `multiple` | `boolean` | Allow multiple files |
| `disabled` | `boolean` | Disable drop zone |

## Performance Tips

1. Use `React.memo` for animated components in lists
2. Set `will-change` CSS property for complex animations
3. Use `transform` and `opacity` for best performance
4. Disable animations for reduced motion preference:

```tsx
const prefersReducedMotion = window.matchMedia(
  '(prefers-reduced-motion: reduce)'
).matches;

<FadeIn disabled={prefersReducedMotion}>
  Content
</FadeIn>
```
